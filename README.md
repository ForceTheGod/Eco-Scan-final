#EcoScan AI

**EcoScan AI** is a web application that lets you upload images or use your device camera for real-time scanning to detect the type of waste and provide **proper disposal instructions**.

By helping users make the right decision *before throwing the trash*, EcoScan AI promotes better recycling habits and contributes to the fight against climate change.

##Features

-  Upload images to classify waste  
-  Real-time camera-based waste detection  
-  AI-powered waste classification  
-  Clear disposal instructions for each waste type  
-  Encourages sustainable and responsible waste management
